Entendendo o Botão de Energia

O botão de energia atua como um interruptor, permitindo a entrada da corrente elétrica para os componentes da máquina. Isso permite a entrada de energia, que passa pela fonte de alimentação, passando pelas conversões necessárias, até chegar a todos os componentes com a voltagem correta. A BIOS/UEFI assume o controle, realiza o POST (Power On Self Test), verifica se os componentes estão funcionando adequadamente e procura um dispositivo de inicialização. O bootloader carrega o sistema operacional e, finalmente, o sistema inicia para o uso.
